Filtering wants by category

The expanded category menu on both platforms: a popover anchored to the trigger on desktop, a bottom sheet on phone. Same control — multi-select with OR — framed to each.

Multi-select · OR. A want has exactly one category, so ticking several shows wants in any chosen one (an AND would always be empty). The trigger shows the count of active filters; Clear resets to all. Only user categories appear — the system/domain ones (Books · Board Games · Video Games) are link-derived and stay out of the manual picker (ADR-0024).
