Starts as a single line — grows only if you need it

The dialog opens as a one-line jot: the amount is the hero, with what-it-was, category and date beneath. Tap ＋ Add another line and it promotes to the multi-line view — your first entry drops into an editable list, the hero becomes the running total, and a composer adds the rest (ADR-0019 line-items). One entry is one currency (ADR-0019). Commit enqueues a single FlowRecorded with its Lines[] to the outbox at POST /months/{y}/{m}/transactions — the date picks the period. Most spends never leave single-line.
