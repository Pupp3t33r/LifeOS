“Ongoing” creation sheet

Every interaction for the Live kind — money that keeps coming until you stop it. It's the add-entry sheet (direction toggle inside, single amount that grows into a multiline list) plus one Repeats block. Ends is only Never or On a date — anything countable is a Payment plan, so this sheet never turns into a schedule. The word “recurring” is gone from the UI — this kind is Ongoing.

Opening → filled
Opens on the hero amount, same as Add. Direction is a toggle inside — never a FAB choice.
