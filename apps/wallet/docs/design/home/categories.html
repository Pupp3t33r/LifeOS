Category colours — the palette + where you assign them

A curated 12-colour palette, not a free colour wheel (that's how you get garish). Every colour is a muted mid-tone in the Calm family with a separate light + dark rendering, so the proportion bars read in both modes. You assign them in Settings → Categories — including recolouring the locked system categories from other domains.

Decisions baked in: 12 fixed slots (enough to stay distinct, few enough to stay calm); light/dark defined per colour; system category names stay locked (domain constants, ADR-0024) but colour is a personal display setting you can recolour; the colour is a client/display concern, not domain data. New categories get an auto-assigned next-free colour, editable. With this in place, Home rows switch to the desaturated proportion bar (panel 4).
