Row spec — every row: count + proportion bar, expandable

Uniform anatomy: icon · name · [bar + count] · amount. Multi-line rows carry a chevron and expand to their lines. Two special cases below: a single-item row (solid bar, no count, no chevron) and a multi-item / single-category row (solid bar, but still expandable).

The rule, finalised:
• Every row = icon · name · [bar + count] · amount.
• Multi-line → chevron; bar shows the category split; count = "N items"; tap expands to per-line rows (dot · name · category · amount).
• Multi-line, single category → solid bar; count = "N items · Category"; still expandable (you may want to see the lines).
• Single line → solid bar; no count, shows the category name instead; no chevron (nothing to reveal — tapping just opens edit).
