Settings — a searchable, grouped page

A single scroll of grouped settings (IDE/Discord-style): a filter narrows by name, and a contents rail on wide screens jumps to — and highlights — the group in view. Only settings with a working write path ship: theme + language (device-local), display currency + month-start (Money ADR-0013), unit system (ADR-0036), the biometric app-lock (native only, ADR-0014), and two pushed surfaces — Categories (ADR-0024/0008) and Exchange rates (ADR-0015). Account name / opening balance are absent — immutable server-side, no edit endpoint.
